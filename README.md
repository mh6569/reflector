# Reflector — Quick Start

## Demo
- `[demo](https://youtu.be/-XQZkCJKQoY)`

## Prerequisites
- Node.js 18+
- npm (or pnpm/yarn if you prefer)

## Install & Run
```bash
npm install
npm run dev
```
Open the printed localhost URL.

## What this demo does
- Loads mock journal entries.
- Lets you add an entry; tags it with sentiment and semantic themes.
- Generates concise weekly insights from your entries (template-based, deterministic).
- Shows a context-aware prompt above the entry box.

## Troubleshooting
- First run downloads small models; keep the tab open until they cache.
- If models won’t load (offline/blocked), retry with network or bundle models locally.
